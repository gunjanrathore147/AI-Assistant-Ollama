# AI-Assistant-Ollama
A local AI assistant built using Python, Tkinter, Ollama and Gemma.
# 🤖 Local AI Assistant

This project is a simple AI Assistant built using:

- Python
- Tkinter
- Ollama
- Gemma 3

## Features

- Local LLM (No API)
- GUI Interface
- Conversation History
- Dark Theme
- Enter Key Support

## Tech Stack

Python

Tkinter

Ollama

Gemma 3
